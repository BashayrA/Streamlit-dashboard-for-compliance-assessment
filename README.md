# Streamlit-dashboard-for-compliance-assessment

This dashboard provides an automated way to analyze Excel datasets related to companies, badges, regions, statuses, and other fields. It reads any uploaded Excel file, cleans the data, detects important columns, and generates clear analytical views without requiring manual configuration.

# How to use 

- Upload Page
This page allows the user to upload an Excel file.
Once the file is uploaded, the dashboard automatically:

Reads all sheets

Cleans the data

Detects column roles such as company, category, region, status, date, and numeric fields

Prepares the dataset for analysis

- Enjoy exploring the dashboard!
